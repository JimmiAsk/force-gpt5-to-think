Force GPT-5 to think

Appends a thinking reminder to your ChatGPT prompt only when the selected model is exactly GPT‑5.

  1. Install Tampermonkey from the Chrome Web Store
  2. Click the button below to install the userscript:
    Install Userscript
  3. Open chatgpt.com and use GPT‑5. The reminder will append on send.

If the button doesn’t work, open the raw userscript URL directly:

https://raw.githubusercontent.com/JimmiAsk/force-gpt5-to-think/main/userscript/force-gpt5.user.js
